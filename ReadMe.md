# IMC Prosperity 4

Resources, setup notes, and tools for our **IMC Prosperity 4** trading competition workflow.

## Quick Start

Activate the virtual environment:

```bash
source prosperity_4/bin/activate
```

## Main Websites

- **Competition site:** https://prosperity.imc.com/game
- **Prosperity 4 Wiki:** https://imc-prosperity.notion.site/prosperity-4-wiki
- **Reference repository:** https://github.com/TimoDiehm/imc-prosperity-3
- **Strategy write-up:** https://medium.com/@td.timodiehm/how-to-almost-win-against-thousands-of-other-teams-in-competitive-environments-bc31387e4b26

## Data Visualization Tool

Use this site to inspect product price graphs, bids, asks, and trades:

- **Dashboard:** https://hwalbertseo.github.io/prosperity_4/

### How to use it

1. Download the data ZIP file from the Prosperity website.
2. Upload that ZIP file into the dashboard.
3. Explore price movements, order book activity, and trade history.

## Notes

- The **Wiki** is especially important and should be one of the main references while working.
- The dashboard is useful for quickly reviewing market behavior from downloaded round data.
